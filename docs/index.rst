===
Mau
===

**Mau** is a lightweight a lightweight markup language heavily inspired by AsciiDoc_, Asciidoctor_ and Markdown_.

The documentation hasn't been converted to reStructuredText yet, and it is hosted at `The Digital Cat Books <https://www.thedigitalcatbooks.com>`_.
